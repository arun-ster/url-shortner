Steps to be followed before running the application:-

1. This repository contains source and sql files needed to run the application. Please copy src.zip and unzip the files to your webroot. 
2. Create a database and edit config/database.php with respect to created database and server details.
3. Edit config/app.php and set url to your respective server url.
4. Run the application with your root address.

Tools Used:-

Server Side :- PHP with Laravel 5 Framework.

Client Side : - Javascript with Jquery, Bootstrap

Css3

HTML5

IDE: Netbeans.
